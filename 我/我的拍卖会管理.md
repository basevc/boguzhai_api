```

sessionid=1h283019400031n22hhu22313145252
auctionStatus=1
```

###设置拍卖会代理出价
Http Request: 

```
m=setAuctionProxyPrice
sessionid=1h283019400031n22hhu22313145252
auctionId=AC1001212
useProxy=1 #1启用代理出价，0不启用代理出价
upperLimit=20000.00 #代理出价上限
```

Http Response:

- code=0 : 设置代理出价成功
- code=-1 : sessionid 错误
- code=1 : 由于其它原因，设置代理出价失败

``` 
{ 
    "code": 0;
    "data": { 
    }
} 
```
