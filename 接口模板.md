用户登录接口
---

### Http 请求 

**请求参数列表**

``` 
m=login
phone=13812345678  #11位手机号码
password=123456   #密码(暂不考虑加密)
```

### Http 回复  

- code=0 : 登录成功，返回用户信息和seesionid
- code=1 : 登录失败，无此用户名
- code=2 : 登录失败，密码错误

**几种情况如下**

