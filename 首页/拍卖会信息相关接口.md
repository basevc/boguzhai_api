### 获取首页拍卖会专场列表

Http Request: 

```
m=getMainauctionPartList
```
Http Response:

- code=0 : 获取信息成功，返回拍卖会列表
- code=1 : 获取信息失败，服务器错误

``` 
{ 
    "code": 0;
    "data": {
    	"auctionPartList":[
    		{
    		"auctionPartId": ""; #拍卖会专场ID
    		"name": "2015年新春大拍 - 字画专场"; #名称由拍卖会+专场组成
    		"image": "http://www.../...png"; #专场图片
    		},
    		...,
    		{
    		"auctionPartId": "";
    		"name": "2015年圣诞大拍 - 古董专场";
    		"image": "http://www.../...png";
    		}
    	]
	}
}
```
