获取首页拍品列表
---

### Http 请求 

**请求参数列表**

- m=getMainLotList
 
---
### Http 回复  

- code=0 : 获取信息成功，返回拍品列表
- code=1 : 获取信息失败，服务器错误

**code=0时 情况如下**

``` 
{ 
    "code": 0;
    "data": {
    	"lotIdList":["ISBN 7-9012-987-4/G·234", "ISBN 7-9012-997-4/G·233","ISBN 7-9012-980-4/H·235"]
	}
}
# 注意：拍品id采用拍品图录号作编码
```
